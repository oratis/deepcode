DeepCode · Visual Design v0.4

Claude Code 的形，DeepSeek 的魂。

面向开发者的本地 AI 编程代理 — 整体复刻 Claude Code 全部能力，底层全面切换到 DeepSeek。

两种形态 · 一个内核：npm i -g deepcode-cli CLI / DeepCode.dmg Mac 客户端

01 · 设计原则

克制、对齐、可读

DeepCode 不是一个"AI 工具"，它是一个长期陪伴写代码的环境。视觉上向 Claude Code 的克制风格看齐，色彩仅在"AI 思考 / 工具执行 / 状态变化"这几个关键瞬间发声。

原则 A · 对齐熟悉感

Claude Code 用户切到 DeepCode，不应有任何重新学习成本。slash 命令、工具卡片、approval 流程、会话侧栏的布局位置、键盘快捷键，全部保持一致。差异仅体现在：品牌色（蓝）、模型名（DeepSeek）、文件名（DEEPCODE.md）。

原则 B · 终端美学优先

Mac 客户端本质是"一个长得好看的终端 + 编辑器"，而不是聊天 App。等宽字体、行号、ANSI 色、diff 视图是一等公民；emoji 和插画退到零；动效只在数据真正变化时出现。

02 · 色板

主色取自 DeepSeek 品牌蓝

蓝是身份色，仅用于 logo、活跃态、AI 输出的边缘点缀。绿（mint）用于"运行中 / 工具成功"，红用于报错，黄用于审批待确认。中性灰阶占据 90% 的画面。
